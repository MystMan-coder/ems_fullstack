# Employee Management System (Full Stack)

A Full Stack Employee Management System built using **Spring Boot, React.js, and MySQL**.  
This project performs basic CRUD operations with REST APIs and a modern frontend.

---

## 🚀 Tech Stack

### Backend
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- REST API

### Frontend
- React.js
- Axios
- Bootstrap / Tailwind (if used)

---

## 📌 Features

- Add new employee
- View all employees
- Get employee by ID
- Update employee details
- Delete employee
- RESTful API integration
- Exception handling (custom errors)
